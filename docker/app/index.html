CI/CD Infrastructure Deployment Project

Automated Cloud Deployment Environment

Projektübersicht

Dieses Projekt beschreibt den Aufbau einer automatisierten Cloud-Deployment-Umgebung. Ziel ist eine reproduzierbare Infrastruktur mit automatisierter Konfiguration, Containerisierung und Bereitstellung von Anwendungen.

Project Overview

This project describes the implementation of an automated cloud deployment environment. The objective is a reproducible infrastructure with automated configuration, containerization and application deployment.
