# IONOS Cloud Infrastructure Automation Demo

## Purpose

This project demonstrates automated cloud infrastructure deployment using Infrastructure as Code principles.

The goal is to build a repeatable environment similar to enterprise cloud deployments.

## Technologies

- IONOS Cloud
- Terraform
- Ansible
- Docker
- Linux Ubuntu
- Git/GitHub

## Current Status

Completed:

- Automated IONOS datacenter creation
- Network configuration
- Public/private LAN setup
- Virtual machine provisioning
- SSH key authentication
- Docker installation

## Architecture

Terraform provisions the infrastructure.

Ansible will configure servers.

Docker will run application services.

## Future Improvements

- CI/CD pipeline
- Monitoring
- Backup automation
